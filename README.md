# Letuscrack-IDE
An Online IDE that compiles various languages like C, C++, Java, JavaScript, TypeScript, Scala, Python, SQL, Go lang, and other different languages based on React Js.

## Dev

### Install dependencies

`npm install`

### Build

`npm run build`
